Enviswitch

Google Chrome Plugin allowing you to change between website subdomains.

Download as .zip Download as .tar.gz View on GitHub

The basic idea behind it is that you can set up a website folder which will then have multiple site environments added to it. These environments should, ideally, be for the same domain with a different sub domain e.g. www.testsite.com, test.testsite.com, dev.testsite.com, etc...

When installed open up the options page of the extension and you should see a box with EnviSwitch Root in it, right click on that to get a menu to add a new site folder.

When adding a new site folder you want to set up the name and site address, the site address would be the domain name only e.g. testsite.com not http://www.testsite.com or www.testsite.com.

Once you have the site folder set up you can then add sites to it by right clicking on the folder and selecting add. You will get a prompt similar to the other which will take the site name and the url for the site. In the site url you want to enter the full url for the site e.g. http://www.testsite.com or http://test.testsite.com.

You can also edit and delete sites and site folders by right clicking on them and selecting the appropriate option.
