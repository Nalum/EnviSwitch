EnviSwitch(Environtment Switch).

I have been looking for something to write for a while and thought this would be a good first attempt. The basic idea behind it is that you can set up a website folder which will then have multiple sites added to it. These sites should, idealy, be for the same domain e.g. www.testsite.com, test.testsite.com, dev.testsite.com, etc...

Designed and Developed by Nalum with the following:

  • jQuery 1.4.2
  • jQuery UI 1.8.1 & EnviSwitch Theme
  • Dynatree 0.5.3
  • Context Menu 1.01
  • Silk Icons 1.3
